# 🌌 GameVault - Cyber-Aurora Marketplace

[![Live Demo](https://img.shields.io/badge/demo-live-00f0ff?style=for-the-badge&logo=vercel)](https://game-vault-psi.vercel.app)
[![API Status](https://img.shields.io/badge/api-active-ff007f?style=for-the-badge&logo=render)](https://gamevault-api-h2i2.onrender.com)

GameVault is a high-performance, full-stack gaming marketplace built with a proprietary **Cyber-Aurora** aesthetic. It features a seamless user experience, real-time database integration, and a premium glassmorphic interface.

## 🚀 Live Links
- **Storefront:** [https://game-vault-psi.vercel.app](https://game-vault-psi.vercel.app)
- **API Documentation:** [https://gamevault-api-h2i2.onrender.com](https://gamevault-api-h2i2.onrender.com)

---

## ✨ Key Features
- **Authentication System**: Secure JWT-based Login and Signup with password hashing.
- **Persistent Cart**: Global state management for shopping carts with local storage persistence.
- **Cyber-Aurora UI**: Premium design system utilizing Glassmorphism, neon accents, and fluid animations.
- **Digital Library**: Dynamic user profiles that track purchased games and recent activity.
- **Performance Optimized**: Implemented Code Splitting (Lazy Loading) for ultra-fast initial page loads.

## 🛠️ Tech Stack
- **Frontend**: React (Vite), React Router 7, Axios, Lucide React.
- **Backend**: Node.js, Express, JWT, Bcrypt.js.
- **Database**: MongoDB (Mongoose).
- **Styling**: Vanilla CSS (Custom Design System).

## 💻 Local Setup

### 1. Clone the repository
```bash
git clone https://github.com/vishal9814/GameVault.git
cd GameVault
```

### 2. Backend Setup
```bash
cd backend
npm install
# Create a .env file with MONGODB_URI and JWT_SECRET
npm start
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## 🎨 Design Philosophy
The **Cyber-Aurora** aesthetic is built on:
- **Depth**: Multi-layered backdrop blurs (`backdrop-filter: blur(16px)`).
- **Vibrancy**: Neon Cyan (#00f0ff) and Magenta (#ff007f) gradients.
- **Typography**: Clean, modern headers using *Space Grotesk* and body text using *Outfit*.
